### 离线部署

安装`nodejs`和`npm`。

```bash
apt install nodejs npm
```

使用`npm`安装`gitbook`。

```bash
npm install gitbook-cli -g
```

克隆此仓库。

```bash
git clone https://github.com/0xf4n9x/Web_Security_Academy-zh
```

使用`gitbook`命令进行部署。

```bash
cd Web_Security_Academy-zh && gitbook serve
```

访问：http://127.0.0.1:4000
